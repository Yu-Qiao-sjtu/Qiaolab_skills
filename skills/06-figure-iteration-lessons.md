# 论文图设计迭代教训总结

## 一、UMAP 散点图

### 迭代路径
1. 初版：14 色散点 + 右侧基础/临床面板 → 右侧无点（中文 label 匹配失败）
2. → 修复匹配 + 左侧标签太小(7pt) → 加质心标签
3. → 中文→英文 + 标签遮挡点 → 标签改图例外置
4. → 不要右侧图 → 仅左侧，XY 轴字体放大
5. → 尝试虚线凸包边界 → 太乱 → 去掉
6. → 点太小(s=4) → s=8 → s=30
7. → 标签与点重叠 → 标签放右侧图例
8. → 图例字号太小 → 22pt bold（与 Title 一致）

### 最终参数
- s=30, alpha=0.6
- 标签：右侧图例，22pt bold，markerscale 1.2
- 轴标签：22pt bold，刻度：15pt
- 图幅：20×13 inch，全英文

### 教训
- 质心标签永远会遮挡数据点 → 用外部图例
- 虚线凸包在密集区域会变成灾难 → 别用
- 默认 s=4 在论文图里太模糊 → s=30 起步
- 中文在 matplotlib 默认字体下大概率乱码 → 英文优先

## 二、上升新信号色条图

### 迭代路径
1. "用什么图展示最强新信号" → 推荐棒棒糖图
2. → "太低级" → Polar Bar / Dumbbell / Arc Diagram
3. → 选"论文" → Dumbbell Chart
4. → 年份颜色不明显(浅灰 vs 深灰) → 蓝 vs 品红
5. → "变丑了，为什么 2026 比 2025 小？" → **筛选逻辑错误！**
   突发术语 ≠ 上升信号。需要 2026 > 2025。
6. → 保留上升的 → 标签重叠 → 去类别标注，纯双色
7. → "能否做色块" → 色条矩阵
8. → 25 个 → "太狭窄" → 50 个
9. → "太丑了" → 方形→圆角色条，亮红→砖红暖色，暖灰背景
10. → "太多了" → 15 个，去方法学通用词
11. → 标签截断 → 左 margin 38%, xlim=-8, 右对齐

### 最终参数
- 圆角色条（FancyBboxPatch），砖红暖色 + 暖灰
- 15 术语，2022-2024=0 且 2026>2025
- 图幅 14×9 inch，左 margin 38%
- 标签 13pt 右对齐，白字标频次
- Times New Roman，600 dpi

### 教训
- "高级"不是更复杂的图表类型，而是更精致的审美
- 灰+黑 Tufte 极简 > 蓝+品红
- 色条矩阵是展示稀疏二维数据的最佳选择
- **筛选逻辑错误是致命伤** —— 先确认数据含义再作图
- FancyBboxPatch 圆角色条比方形好看一个数量级

## 三、Streamgraph

### 迭代路径
1. 14 大类 × 25 年 → 右侧 13 标签严重重叠
2. → 双列交错 → "还是不行" + Brain Regions(8410)碾压
3. → 去 Brain Regions + Noise → 12 个有意义大类
4. → 单列标签 + 灰色引导线 → 重叠解决
5. → 颜色相近 → distinctipy CIELAB 11 色 + 排除灰色
6. → DPI 不够 → 600→1200
7. → 标签太小(12.5pt) → 18pt
8. → X 轴刻度太小(9pt) → 18pt
9. → 无标签版 → 另存不覆盖

### 最终参数
- 12 主题，去杂项+噪声
- 30×12 inch，1200 dpi
- distinctipy CIELAB 色，排除灰色
- 单列右置标签，等距引导线
- 标题 24pt，标签/X 轴 18pt，白边分隔

### 教训
- 一个巨无霸层会压扁所有其他层 → 必须去掉
- 单列 > 双列（用户期望垂直对齐阅读）
- A4 打印需 18pt+ （屏幕看不出来）
- 同时出有标签+无标签版

## 通用设计铁律

| 铁律 | 详情 |
|------|------|
| **先问用途** | 论文正文 / 补充材料 / 海报 / PPT → 参数不同 |
| **先问画幅** | A4 纸 vs 屏幕 → DPI 和字号差距巨大 |
| **先问语言** | 中文 vs 英文 → matplotlib 字体差异可毁图 |
| **筛选逻辑 > 可视化** | 数据错再好看的图也废 |
| **去杂项口袋** | 体量碾压的层必须单独处理 |
| **CIELAB 配色** | distinctipy > ColorBrewer > 默认 tab10 |
| **DPI** | 屏幕 300 / 投稿 600 / 海报 1200 |
| **字体** | 正文图 18pt min，A4 同理 |
| **标签不重叠** | 外部图例/引导线 > 质心标签 > 双列 |
| **无标签版必出** | 用户可能需自己排版 |
| **色条 > 圆点** | 稀疏二维数据（术语×年份）色条矩阵更直观 |
